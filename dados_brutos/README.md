Pasta destinada aos dados originais utilizados no projeto.

# Dados Brutos

Esta pasta contém os arquivos originais utilizados no processo de ETL.

## Arquivos

### fornecedores.csv
Contém informações cadastrais dos fornecedores.

### lojas.txt
Lista de lojas com identificação e localização UF.

### metas.xlsx
Metas de vendas por período, categoria e loja.

### produtos.xlsx
Cadastro de produtos.

### Pedidos.pasta
Arquivos individuais de pedidos por ponto de venda carregado por pasta, posteriormente unificados no Power Query.
